# HIDRA_S.A_de_C.V

## 📌 Descripción

Este proyecto consiste en el desarrollo de un sistema web administrativo en **PHP puro** para la gestión de una empresa de agua potable.

El sistema permitirá controlar de forma integral:

- Clientes (naturales y jurídicos)
- Facturación mensual
- Pagos y abonos
- Moras configurables
- Lecturas de medidor
- Sectores y conexiones
- Reportes administrativos
- Configuración del sistema

El enfoque del sistema es ser **completo, organizado y fácil de usar**, manteniendo una estructura simple sin uso de frameworks.

---

## 🎯 Objetivo

Desarrollar una plataforma que permita:

- Automatizar la facturación mensual
- Controlar pagos y deudas
- Gestionar usuarios por sector
- Visualizar el estado del servicio
- Generar reportes administrativos
- Escalar el sistema en el futuro

---

## 🧱 Arquitectura del Proyecto

El sistema utiliza una estructura basada en **PHP puro con separación por responsabilidades**, similar a un modelo MVC simplificado.
